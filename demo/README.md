# 合规库管理 Demo 演示

基于《合规库管理需求-v4》打造的演示应用，覆盖完整操作流程。

## 操作流程

```mermaid
flowchart LR
    A[上传文件] --> B[拆分检查项]
    B --> C[创建合规检查任务]
    C --> D[执行合规检查任务]
    D --> E[下载合规检查报告]
```

## 功能模块

| 模块 | 功能点 |
|------|--------|
| **合规库管理** | 网格展示、点击查看文件、上传文件 |
| **检查项管理** | 列表展示、所属合规文件列 |
| **检查任务** | 网格列表、新建任务、执行检查、停止、删除、下载报告、检查详情 |

## 运行方式

### 方式一：直接打开

```bash
# 在浏览器中打开
open demo/index.html
# 或双击 index.html
```

### 方式二：本地服务（推荐）

```bash
cd demo
python3 -m http.server 8080
# 访问 http://localhost:8080
```

或使用 npx：

```bash
cd demo
npx serve .
# 访问提示的地址
```

## 演示说明

1. **合规库管理**：点击任意文件卡片可查看详情；点击「上传文件」可添加新文件
2. **检查项管理**：查看检查项与所属合规文件的对应关系
3. **检查任务**：
   - 新建任务：配置任务名称(50字符)、选择资产、勾选检查项、备注
   - 对「检查中」任务：点击「合规检查」填写结果并结束；点击「停止检查」可中止
   - 对「已完成」任务：点击「下载报告」或「检查详情」查看
   - 检查详情含：任务信息、检查结果、用户上传附件
